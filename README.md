# Image-Manipulation-over-voice-commands
voice commands are used for basic image editibng

# Installing Packages
-  # Tkinter
-       pip install tkinter
- # Pillow
-       pip install pillow
- # Pygame
-       pip install pygame
- # NLTK
-       pip install nltk
- # IPython
-       pip install ipython
- # Speech Recognition
-       pip install speechrecognition
- # Pyaudio
-       pip install pyaudio


# Running the Program
1) Extract the files in a folder
2) Execute pyhton file _elements.py_
